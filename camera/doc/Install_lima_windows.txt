Erreur execution install.bat:
Dans install.bat le script de compilation de la caméra Basler est le suivant:

rem compilation des plugins des cameras actives dans le fichier config.inc
for /f "delims=:" %%i in ('type %CurrentPath%\config.inc') do (
	set ligne=%%i
	set ligne_temp=%%i
	if "!ligne:~0,7!" == "COMPILE" (
		rem call:strlen longueur !ligne! 
		rem echo longueur : !longueur!
		if "!ligne:~-1!" == "1" (
			if not "!ligne:~8,-2!" == "CORE" (
				rem compilation du plugin active
				cd /D !CurrentPath!\camera\!ligne:~8,-2!\build\msvc\9.0\lib!ligne:~8,-2!
				msbuild.exe Lib!ligne:~8,-2!.sln /t:build /fl /flp:logfile=MyProjectOutput.log /p:Configuration=Release;Plateform=Win32 /v:d
			)
		)
	)
)

Problème car la solution dans !CurrentPath!\camera\!ligne:~8,-2!\build\msvc\9.0\libBasler ne s'appelle pas  "LibBasler.sln" mais "LibLimaBasler.sln"
Il faut donc soit renommer la solution en "LibBasler.sln"  soit modifier install.bat et mettre "msbuild.exe LibLima!ligne:~8,-2!.sln"

Après modif de install.bat les librairies Basler sont copiées dans "C:\Python27\Lib\site-packages\Lima" (void compile.txt)
